\section{結論}

本研究では，変数空間における変数間関係性の分析を支援するためのビジュアル分析システムを開発し，加熱球まわりの熱流体計算結果に対して適用した．以下に本研究で得られた結論をまとめる．
\begin{itemize}
\item 提案システムを利用することで渦と熱輸送の間の因果関係を分析可能である．
\item 提案システムにより対話的かつ連続的に分析可能である．
\end{itemize}

伝熱工学の専門家が熱流体解析結果から得られる第二不変量と熱流束量との関係の時系列変化を本システムで可視化することにより，本実験での分析支援において，重要な役割を果たすことが明らかとなった．

\section{課題}

本研究で得られた結論を踏まえて，今後検討するべき課題を以下にまとめる．
\begin{itemize}
\item 3つ以上の変数間の関係性を分析する機能の実装
\item 操作性のさらなる向上
\end{itemize}

本システムでは因果関係を分析する領域の絞り込みを変数空間可視化サブシステムから行う場合，決められた２変数を元に絞りこみを行う．しかし，本システムでは３変数以上の統計情報を元に絞り込みを行うことができない．３変数以上の統計情報を使うことが出来れば，例えば温度勾配の変動距離，第二不変量の変動距離，データの持つｘ座標の３つを変数空間可視化サブシステム上で表現できれば，物理空間可視化サブシステム上での操作なしに，大まかな領域ごとの統計情報を把握することが可能となり，領域の絞り込みの手間を減らすことが可能となる．

専門家に対するインタビュー結果である図\ref{fig:7.2}より，操作性に難があることが明らかとなった．操作性はビジュアル分析システムの対話性につながる要素であり，操作性の向上は本システムの特徴をより良くすることができる点で重要である．操作性を向上させるためには例えば，操作をより直感的にする，システム上のボタンの数を減らす，といったことができる．
